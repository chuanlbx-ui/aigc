**路由逻辑**:
```typescript
private getAdapter(platformName: string): PlatformAdapter | null {
  switch (platformName.toLowerCase()) {
    case 'weibo':
    case '微博':
      return new WeiboAdapter();
    case 'bilibili':
    case 'b站':
      return new BilibiliAdapter();
    case 'douyin':
    case '抖音':
      return new DouyinAdapter();
    case 'kuaishou':
    case '快手':
      return new KuaishouAdapter();
    default:
      return null;
  }
}
```

### 1. 统一的适配器接口

所有平台适配器继承自 `PlatformAdapter` 基类，提供：

- 通用的 DOM 等待工具（`waitForElement`）
